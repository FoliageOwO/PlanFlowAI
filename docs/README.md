## 阅读建议

```
新加入项目 → README.md（项目概览）
             → REQUIREMENTS.md（理解需求）
             → ARCHITECTURE.md（理解架构）
             → ROADMAP.md（了解开发节奏）

着手开发 → PROJECT.md（数据库 + API 细节）
         → ROADMAP.md（当前阶段 TODO）

答辩评审 → ARCHITECTURE.md（架构亮点）
         → PROJECT.md（设计深度）
         → 演示脚本（PROJECT.md 附录）
```
